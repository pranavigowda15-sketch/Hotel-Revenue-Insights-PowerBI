# Hotel Revenue Insights Dashboard

## Overview
The Hotel Revenue Insights Dashboard is a data analytics project designed to analyze and visualize hotel revenue performance across multiple dimensions such as cities, room types, booking platforms, and seasons. The project helps stakeholders understand revenue trends, identify high-performing segments, and make data-driven decisions for pricing and business strategy.

---

## Features
- Revenue analysis across cities and hotels
- Room type and booking platform performance comparison
- Seasonality and trend analysis
- Identification of high-revenue segments
- Interactive and user-friendly Power BI dashboard
- Data-driven insights for pricing and occupancy optimization

---

## Technologies Used
- Power BI
- SQL
- Power Query
- Microsoft Excel
- Data Modeling (Star Schema)
- Git & GitHub

---

## Dataset Description
The dataset contains hotel booking and revenue-related information, including:
- City and hotel details
- Room categories
- Booking platforms
- Revenue metrics
- Seasonal and time-based data

The data was cleaned, transformed, and modeled using Power Query before visualization.

---

## How It Works
1. Raw hotel booking data is loaded into Power BI.
2. Data cleaning and transformation are performed using Power Query.
3. A star schema data model is designed for efficient analysis.
4. Key revenue metrics and KPIs are calculated.
5. Interactive dashboards are built to explore revenue trends and patterns.
